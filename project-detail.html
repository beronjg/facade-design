The Challenge

The building faces South-West, receiving intense solar radiation between 2 PM and 5 PM. A standard glass facade would result in a huge energy load (cooling costs). The client wanted a "Transparent" look but with "Solid" performance.

Problem: How do we block sun without blocking the view?

Parametric Logic

def calculate_rotation(sun_vector, normal):

// Calculate angle based on Azimuth

angle = Vector3.Angle(sun_vector, normal)

if angle > 45:

rotation = 90 # Full Close

else:

rotation = remap(angle, 0, 45, 0, 90)

return rotation
